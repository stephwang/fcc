URL shortener microservice

Pass a URL as a parameter and you'll receive a shortened URL in the JSON response.

Example usage:

https://little-url.herokuapp.com/new/https://www.google.com

Example output:

{ "original_url":"https://www.google.com", "short_url":"https://little-url.herokuapp.com/8170" }

How to use shortened URL

The given short_url (example: https://little-url.herokuapp.com/8170) will redirect to the original URL you provided (example: https://www.google.com)
